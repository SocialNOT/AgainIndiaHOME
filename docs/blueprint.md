# **App Name**: Again India

## Core Features:

- Sankhya Core AI Engine: A Gemini-powered reasoning tool integrating Pythagorean/Chaldean Numerology, Vedic Astrology (Jyotish), and Vastu Shastra principles to generate profound insights.
- Real-time Contextual Data Synchro: Automatically retrieves precise user location (Google Maps API) and current astronomical data (Google Astronomy API or similar ephemeris) to provide real-time planetary positions and other temporal context to Sankhya.
- Loom Multi-Domain Analysis Tool: An AI tool that dynamically cross-references the user's profile with real-time planetary transits and Vastu insights based on their current GPS location, creating a multi-dimensional personal analysis.
- Personalized Micro-Rituals Generator: An AI tool that produces bespoke daily 'Micro-Rituals' (e.g., specific colors to wear, frequency music recommendations) as personalized remedies based on Sankhya's analytical outputs.
- Predictive Life-Graph Visualization: A dynamic, scrollable timeline visualizer presenting 'peak energy periods' and significant life transits derived from Sankhya's predictive analytics, giving a 'Life-Graph'.
- Sankhya Direct Query Interface: An interactive interface for users to directly ask the Sankhya AI tool specific questions or provide real-time scenarios (e.g., 'Sankhya, check Vastu for this new property') to receive instant AI-driven analysis via text or voice.
- Micro-Ritual Tracking & Vault: A dedicated section for users to view, manage, and track the progress and effectiveness of their assigned 'Micro-Rituals' and recommended practices.

## Style Guidelines:

- Primary color: Deep celestial blue (#3333CC) to evoke universal mathematics and cosmic wisdom, suitable for glowing lines and interactive elements against a dark backdrop.
- Background color: A very dark indigo (#17171C) for an immersive, expansive feel, like navigating deep space.
- Accent color: Bright cyan-blue (#85E6FF) for 'glowing neon lines', interactive prompts, and key highlights, providing strong contrast and a sense of futuristic technology.
- Headline font: 'Space Grotesk' (sans-serif) for its computerized, techy, and scientific feel, aligning with the 'numbers of the universe' theme. Body text font: 'Inter' (sans-serif) for neutral, modern readability for analytical content.
- Utilize minimalist line icons with subtle glowing effects, reflective of 'glowing neon lines' and abstract 'numbers'. Elements should feel integrated into a 3D, fluid environment.
- Adhere to the 'Celestial Orrery' concept: a minimalist top bar (Location/Time), a central 3D workspace featuring a rotating sphere of interactive elements, and a sticky glassmorphic bottom dock for navigation.
- Incorporate fluid particles and dynamic rotating elements, especially for the central 3D sphere. Implement Lottie animations for number displays and haptic feedback for significant user interactions, enhancing the 'hypnotic' and tactile experience.
